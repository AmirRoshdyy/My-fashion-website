# StyleHub - Clothing E-commerce Website

A modern, responsive clothing store built with HTML, CSS, and JavaScript.

## Features
- 🛍️ Product catalog with images
- 📏 Size selection
- 🛒 Shopping cart (saved in localStorage)
- 📱 Fully responsive design
- ⚡ Fast and lightweight (no frameworks)

## How to Run
1. Clone this repo: `git clone https://github.com/YOUR-USERNAME/my-clothing-store.git`
2. Open `index.html` in your browser
3. That's it!
